# portfolio-unity-tutorial-sequence-system
[포트폴리오] 라이브 서비스 Unity-SLG 게임을 위한 상태 기반 튜토리얼·시퀀스 시스템

이 저장소는 모바일 SLG 라이브 서비스 게임에서 사용된
튜토리얼 및 연출 시퀀스 시스템을 포트폴리오 목적으로 정리한 코드입니다.

단순히 한 번 실행되고 끝나는 튜토리얼이 아니라,
재접속, 중단 후 이어하기, 스킵, 조건 분기 등
실제 라이브 서비스 환경에서 발생하는 상황을 고려해
상태 기반(State-driven) 구조로 설계되었습니다.

튜토리얼 콘텐츠를 단발성 기능이 아닌
확장 가능하고 안정적으로 운영 가능한 시스템으로 관리하는 것을 목표로 하였으며,
실서비스 운영 과정에서의 안정성과 유지보수를 중점적으로 고려한 설계 예시를 담고 있습니다.

TutorialManager
튜토리얼 전체 생명주기를 관리하며, 서버에 저장된 진행 상태를 기반으로 재접속 시 이어하기 또는 종료 여부를 판단.

SequenceManager
현재 실행할 시퀀스를 선택하고, 스킵·중단·재개 등 라이브 서비스 상황에서 필요한 흐름 제어를 담당.

SequenceControllerRoutine
코루틴을 이용해 각 노드를 순차적으로 실행, 조건 분기나 스킵 발생 시에도 흐름이 꼬이지 않도록 상태를 관리.

Sequence Node
스테이지 전환, 다이얼로그 표시, 조건 체크 등 튜토리얼을 구성하는 최소 실행 단위.
